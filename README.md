
## Getting Started

First, run the development server:

```bash
npm install 
npm run dev 
```


## Architecture

Built with Next.js (App Router) using a layered structure:

app/ – Routing & layouts

components/ – Reusable UI components

services/ – API calls

hooks/ – Business logic

stores/ – Global state

enums/ & interfaces/ – Shared types

lib/ – Utilities

core/ – Configuraton
